## ✅ 4. 数据库权限复查（04 §T5-3）

**所有表**权限：仅 `appserver` 角色可写，前端永不直连。

```sql
-- 示例：users 表
REVOKE ALL ON users FROM PUBLIC;
GRANT SELECT, INSERT, UPDATE, DELETE ON users TO appserver;
-- 其他表同理
```

Prisma 用专用用户（`appserver`），不用 `postgres` 超级用户。
